# Æther Matter Controller for Android

Æther is a simple standalone [Matter](https://csa-iot.org/all-solutions/matter/) controller for Android.

The app keeps the Matter fabric on your Android phone, so it does not require a hub or other third-party controller hardware.

## Build

From the repository root:

```console
./gradlew :app:assembleDebug
```
